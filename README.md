# Developer Introduction Card

This is a simple **Developer Introduction Card** built using **HTML and CSS**.  
It presents a short profile about me with my skills, interests and contact links.

The card uses a **flip animation** so that:
- The **front side** shows my profile and skills.
- The **back side** contains a brief introduction and links to connect with me.

## Features

- Clean pastel UI design
- CSS flip card animation
- Profile photo section
- Skills badges
- Connect links (GitHub, LinkedIn, Email, LeetCode)
- Fully built with **HTML and CSS only**
- .gitignore to ignore the image

## Technologies Used

- HTML5
- CSS3

## Purpose

This project was created as a small front-end exercise to practice:
- Layout design
- CSS styling
- Animations
- Component-style UI development
